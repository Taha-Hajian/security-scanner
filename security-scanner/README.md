# اسکنر امنیت وب‌سایت

پروژه درس امنیت شبکه — یک وب‌اپ ساده با Flask که آدرس یک سایت را می‌گیرد و موارد زیر را بررسی می‌کند:

- فعال بودن HTTPS و ریدایرکت خودکار از HTTP
- اعتبار و تاریخ انقضای گواهی SSL
- هدرهای امنیتی: HSTS، X-Frame-Options، X-Content-Type-Options، CSP، Referrer-Policy، Permissions-Policy
- افشای نسخه سرور در هدر Server
- فلگ‌های Secure و HttpOnly روی کوکی‌ها

در نهایت یک امتیاز امنیتی (۰ تا ۱۰۰) و نتیجه «امن / نیمه‌امن / ناامن» نمایش می‌دهد.

## نحوه اجرا

این پروژه به اینترنت نیاز دارد (چون باید واقعاً به سایت هدف وصل شود)، پس باید روی سیستم خودت اجرایش کنی، نه در این محیط چت.

```bash
# ۱. وارد پوشه پروژه شو
cd security-scanner

# ۲. نصب پیش‌نیازها (ترجیحاً داخل virtualenv)
pip install -r requirements.txt

# ۳. اجرا
python app.py
```

بعد آدرس زیر رو توی مرورگر باز کن:

```
http://127.0.0.1:5000
```

## ساختار پروژه

```
security-scanner/
├── app.py              # منطق اصلی اسکن (HTTPS, SSL, هدرها, کوکی)
├── templates/
│   └── index.html      # رابط کاربری (طراحی شیشه‌ای + فونت Vazirmatn)
├── requirements.txt
└── README.md
```

## نکات برای دفاع از پروژه

- منطق اسکن کاملاً سمت سرور (Python) انجام می‌شود، چون مرورگر به دلایل امنیتی (CORS و عدم دسترسی JS به گواهی SSL) نمی‌تواند این بررسی‌ها را مستقیم انجام دهد — همین نکته خودش یک مفهوم امنیتی خوب برای توضیح در دفاع پروژه است.
- می‌توانی فاکتورهای بیشتری اضافه کنی، مثلاً: بررسی نسخه TLS (۱.۲ در مقابل ۱.۳)، بررسی DNS، یا اسکن چند URL هم‌زمان.
- اگر سایتی فایروال یا محدودیت داشته باشد ممکن است در زمان مناسب پاسخ ندهد (Timeout) — این رفتار طبیعی است.
